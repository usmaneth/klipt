Usage Guidelines

✓ Do

Use the full logo with adequate clear space (minimum 1× the icon height on all sides). Maintain the gradient direction at 135°. Use on dark or light backgrounds.

✗ Don't

Don't rotate, stretch, or skew the logo. Don't change the gradient colors. Don't place on busy backgrounds without a container. Don't capitalize the wordmark.

✓ Do

Use the icon-only mark for favicons, app icons, and small spaces. Keep the pixel grid and broadcast waves visible at all sizes.

✗ Don't

Don't use the icon below 24px. Don't add drop shadows or outlines. Don't recolor the record dot — it's always red. Don't separate the icon elements.

Minimum Sizes
Full logo: Minimum width 120px
Icon only: Minimum 24×24px
Clear space: Minimum padding equal to the height of the icon on all sides
Favicon: Use the simplified icon variant at 16×16px and 32×32px
